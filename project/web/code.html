Key Milestones
in Internet History

1961

Packet switching theory

Leonard Kleinrock publica la primera teoría sobre conmutación de paquetes, el fundamento de la transmisión de datos moderna.

First ARPANET message

Se envía el primer mensaje entre computadoras de UCLA y Stanford, marcando el nacimiento práctico de la red.

1969
1983

TCP/IP adoption

ARPANET cambia oficialmente al protocolo TCP/IP, permitiendo la interconexión de redes dispares (nace Internet).

WWW Proposal

Tim Berners-Lee propone el sistema de hipertexto en el CERN, estableciendo las bases de la World Wide Web.

1989
2006

Modern Mobile Web

Explosión de las redes sociales y la transición masiva a la navegación móvil y aplicaciones web progresivas.
